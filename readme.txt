
==== ITERATION 3 READ ME ====

============
GAMIFICATION
============

Avatars, titles and themes are added as unlockables under the collection tab in the drawer.
Users earn points by completing walks; automatic arrival when destination is reached grants users + 100 points.

For testing, there is "ADD POINTS TEST" button enabled in the collection tab, which will add + 100 points to the user.

Avatar and title will show up in user profile.
Theme will change the color of the toolbar in the entire app.

Leaderboard is in a seperate tab in the drawer


==========
PERMISSION
==========

When user X creates a group while being monitored by user Y, a permission request to lead the group will be sent to user Y, who may accept or reject the request. User X will be able to view the status of permission request he/she made.
